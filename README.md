# Инструкция по работе с командами Git

## 1. **git init** — инициализация локального репозитория.
## 2. **git status** — получение информации о текущем состоянии репозитория.
## 3. **git add** — добавление файлов к следующему коммиту.
## 4. **git commit -m "message"** — создание коммита.
## 5. **git log** — вывод истории коммитов с их хеш-кодами.
## 6. **git puch** - Отправить свою версию репозитория во внешний репозиторий поможет команда.
